NAME

MgetLegend - read a legend

SYNOPSIS

#include "csf.h"

int MgetLegend
(
	MAP *m,
	CSF_LEGEND *l
);

PARAMETERS

MAP *m
Map handle
CSF_LEGEND *l
array large enough to hold name and all entries, the entries are sorted struct CSF_LEGEND is typedef'ed in csfattr.h

DESCRIPTION

MgetLegend reads a version 2 and 1 legend. Version 1 legend are converted to version 2: the first array entry holds an empty string in the description field.

RETURNS

0 if no legend is available or in case of an error, nonzero otherwise
