NAME

RputXUL - change the x value of upper left co-ordinate

SYNOPSIS

#include "csf.h"

REAL8 RputXUL
(
	MAP *map,
	REAL8 xUL
);

PARAMETERS

MAP *map
map handle
REAL8 xUL
new x value of top left co-ordinate

DESCRIPTION

RputXUL changes the x value of upper left co-ordinate.

RETURNS

the new x value of upper left co-ordinate or 0 case of an error.

MERRNO

NOACCESS
