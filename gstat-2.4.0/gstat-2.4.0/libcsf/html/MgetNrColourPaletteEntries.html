NAME

MgetNrColourPaletteEntries - get the number of colour palette entries

SYNOPSIS

#include "csf.h"

size_t MgetNrColourPaletteEntries(MAP *m);

PARAMETERS

MAP *m
the map to get it from

DESCRIPTION

MgetNrColourPaletteEntries returns the number of rgb tupels of the colour palette. Each tupel is a sequence of 3 UINT2 words describing red, green and blue.

RETURNS

the number of rgb tupels, or 0 if not available or in case of error
