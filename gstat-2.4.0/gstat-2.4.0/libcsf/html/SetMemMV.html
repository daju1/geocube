NAME

SetMemMV - set an array of cells to missing value

SYNOPSIS

#include "csf.h"

void SetMemMV
(
	void *buf,
	size_t nrElements,
	CSF_CR cellRepr
);

PARAMETERS

void *buf
write-only buffer with cells
size_t nrElements
number of cells
CSF_CR cellRepr
cell representation

* other version 2 cell representations
  • CR_REAL8 - Double scalar or directional, and also the only type that can hold all cell representation without loss of precision.
