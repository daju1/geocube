NAME

MputHistory - put the history attribute

SYNOPSIS

#include "csf.h"

int MputHistory
(
	MAP *m,
	char *history
);

PARAMETERS

MAP *m
the map to get it from
char *history
the new history string Is a C-string, `\0`-terminated

DESCRIPTION

MputHistory writes the history string to a map. An existing history is overwritten.

RETURNS

0 if not available or in case of error, nonzero otherwise
