NAME

RputAngle - put new angle

SYNOPSIS

#include "csf.h"

REAL8 RputAngle
(
	MAP *map,
	REAL8 angle
);

PARAMETERS

MAP *map
map handle
REAL8 angle
new angle

DESCRIPTION

RputAngle changes the angle of the map.

RETURNS

new angle or -1 in case of an error.

MERRNO

NOACCESS BAD_ANGLE
