NAME

RgetRowCol - compute row, column number of true world co-ordinate

SYNOPSIS

#include "csf.h"

int RgetRowCol
(
	const MAP *m,
	double x,
	double y,
	size_t *row,
	size_t *col
);

PARAMETERS

const MAP *m
map handle
double x
x of true co-ordinate
double y
y of true co-ordinate
size_t *row
write-only. Row number (y-pos). Undefined if (x,y) is outside of map
size_t *col
write-only. Column number (x-pos). Undefined if (x,y) is outside of map

DESCRIPTION

RgetRowCol computes row, column number of true world co-ordinate.

RETURNS

0 if the co-ordinate is outside the map, 1 if inside, -1 in case of an error

MERRNO

ILL_CELLSIZE
