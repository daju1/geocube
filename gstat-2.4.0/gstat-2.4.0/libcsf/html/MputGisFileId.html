NAME

MputGisFileId - put the gis file id

SYNOPSIS

#include "csf.h"

UINT4 MputGisFileId
(
	MAP *map,
	UINT4 gisFileId
);

PARAMETERS

MAP *map
map handle
UINT4 gisFileId
new gis file id

DESCRIPTION

put the gis file id

RETURNS

the "gis file id" field or MV_UINT4 in case of an error

NOACCESS
