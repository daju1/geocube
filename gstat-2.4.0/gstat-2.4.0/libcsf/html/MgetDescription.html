NAME

MgetDescription - get the description attribute

SYNOPSIS

#include "csf.h"

int MgetDescription
(
	MAP *m,
	char *des
);

PARAMETERS

MAP *m
the map to get it from
char *des
the resulting description string

DESCRIPTION

get the description attribute

RETURNS

0 if not available or in case of error, nonzero otherwise
