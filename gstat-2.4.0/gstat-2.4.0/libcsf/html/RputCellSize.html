NAME

RputCellSize - put cell size

SYNOPSIS

#include "csf.h"

REAL8 RputCellSize
(
	MAP *map,
	REAL8 cellSize
);

PARAMETERS

MAP *map
map handle
REAL8 cellSize
new cell size

DESCRIPTION

put cell size

RETURNS

the new cell size or -1 in case of an error.

MERRNO

ILLHANDLE NOACCESS ILL_CELLSIZE
