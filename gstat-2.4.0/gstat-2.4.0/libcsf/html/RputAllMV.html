NAME

RputAllMV - make all cells missing value in map

SYNOPSIS

#include "csf.h"

int RputAllMV(MAP *m);

PARAMETERS

MAP *m
Not Documented.

DESCRIPTION

RputAllMV writes a missing values to all the cells in a map. For this is allocates a buffer to hold one row at a time.

RETURNS

1 if succesfull, 0 in case of an error
