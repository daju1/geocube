NAME

IsMV - test if a value is missing value

SYNOPSIS

#include "csf.h"

int IsMV
(
	const MAP *map,
	const void *cellValue
);

PARAMETERS

const MAP *map
map handle
const void *cellValue
value to be tested

DESCRIPTION

test if a value is missing value

RETURNS

0 if not, nonzero if it is a missing value
